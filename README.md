# Oreos 🦀

A minimal operating system kernel bootstrapped with Rust

## Building

- Install the `x86_64-unknown-none` target using rustup:

  ```console
  rustup target add x86_64-unknown-none
  ```

- Build by running:

  ```console
  cargo build --target x86_64-unknown-none
  ```
